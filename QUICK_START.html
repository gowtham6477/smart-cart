1Enable Annotation Processing

This is the MOST IMPORTANT step to fix Lombok errors:

  1. Open IntelliJ IDEA
  2. Press Ctrl+Alt+S to open Settings
  3. Navigate to: Build, Execution, Deployment ‚Üí Compiler ‚Üí Annotation Processors
  4. ‚úÖ Check the box: Enable annotation processing
  5. Click Apply then OK

2Install Lombok Plugin

Make sure Lombok plugin is installed:

  1. Go to: File ‚Üí Settings ‚Üí Plugins
  2. Search for: Lombok
  3. If not installed, click Install
  4. Restart IntelliJ IDEA if prompted

3Reload Maven Project

Download all dependencies from Maven Central:

  1. Find the Maven tool window (usually on the right side)
  2. If not visible: View ‚Üí Tool Windows ‚Üí Maven
  3. Click the Reload All Maven Projects button (üîÑ circular arrows icon)
  4. Wait patiently! This downloads all libraries (5-15 minutes)
  5. Watch the progress bar at the bottom of IntelliJ

4Build the Project

Compile everything:

  1. Go to: Build ‚Üí Build Project (or press Ctrl+F9)
  2. Wait for the build to complete
  3. Check the Build window at the bottom
  4. All errors should disappear! ‚ú®

5Verify the Fix

Check if everything works:

  1. Open src/main/java/org/example/dto/RegisterRequest.java
  2. All red underlines should be gone
  3. Imports should be recognized
  4. No more "Cannot resolve" errors!
