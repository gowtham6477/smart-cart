2Setup MongoDB Atlas

Why: Your application needs a database to store users, bookings, payments, etc.

Quick Setup:

  1. Sign Up: MongoDB Atlas
  2. Create FREE Cluster:
    • Click "Create" under M0 FREE tier
    • Choose AWS provider
    • Select region closest to you
  3. Create Database User:
    • Username: smartcart_admin
    • Click "Autogenerate Secure Password"
    • ‚ö†Ô∏è SAVE THIS PASSWORD!
  4. Whitelist IP:
    • Add your current IP
    • OR use 0.0.0.0/0 for development
  5. Get Connection String:
    • Click "Connect" ‚Üí "Connect your application"
    • Copy the connection string
    • Replace <password> with your password
  6. Update Config:
    • Open: src/main/resources/application.properties
    • Update MongoDB URI with your connection string
